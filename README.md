# TODO Application

## 概要
シンプルで使いやすいTODOアプリケーションです。

## 技術スタック
- フロントエンド: React + TypeScript
- バックエンド: Node.js + Express + TypeScript
- データベース: PostgreSQL
- インフラ: AWS (ECS, RDS)
- IaC: Terraform
- CI/CD: GitHub Actions

```
todo-app/
```
